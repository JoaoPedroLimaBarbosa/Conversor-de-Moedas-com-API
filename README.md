# 💰 Conversor de Moedas com API

## 📌 Descrição
Este projeto é uma aplicação web simples que consome uma API pública para exibir a cotação de moedas em tempo real.

O usuário pode digitar uma moeda (como USD, EUR ou BTC), e o sistema retorna informações atualizadas em relação ao Real Brasileiro (BRL).

---

## 🚀 Funcionalidades

- Buscar cotação de moedas em tempo real  
- Exibir valor atual da moeda  
- Mostrar valor máximo do dia  
- Mostrar valor mínimo do dia  
- Exibir data da cotação  
- Feedback de carregamento  
- Botão para limpar os dados  

---

## 🧠 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Fetch API  
- JSON  

---

## 🌐 API utilizada

AwesomeAPI (Economia)

Endpoint de exemplo:
https://economia.awesomeapi.com.br/json/last/USD-BRL

## 💡 Como funciona

- O usuário digita uma moeda no campo de entrada  
- O JavaScript captura esse valor  
- A URL da API é montada dinamicamente  
- A requisição é feita usando `fetch()`  
- Os dados retornam em formato JSON  
- As informações são exibidas na tela com `innerHTML`
